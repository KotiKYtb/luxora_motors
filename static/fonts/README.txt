Polices du site
===============

TITRES : Helios Extended
------------------------
Ajoutez ici le fichier de la police Helios Extended pour les titres.

Nom attendu du fichier : HeliosExtended.woff2 (ou HeliosExtended.woff)

Si le fichier a un autre nom (ex. HeliosExt.woff2), renommez-le en HeliosExtended.woff2
ou modifiez les chemins dans static/css/main.css (rechercher @font-face "Helios Extended").

Sans ce fichier, les titres utiliseront la police système sans-serif.


TEXTE : Inter
-------------
La police Inter est chargée via Google Fonts (templates base.html et cms/base_cms.html).
Aucun fichier à ajouter ici.
